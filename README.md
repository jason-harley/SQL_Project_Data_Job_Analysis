# Analysis

### 1. Top-paying Data Analyst jobs:
To identify the highest-paying roles, I filtered data analyst positions by average yearly salary and location, focussing on roles in the UK.

```sql
SELECT
    job_title,
    job_location,
    salary_year_avg AS salary,
    company_dim.name AS company_name
FROM
    job_postings_fact
LEFT JOIN company_dim
ON job_postings_fact.company_id = company_dim.company_id
WHERE
    job_title_short = 'Data Analyst'
    AND job_schedule_type = 'Full-time'
    AND salary_year_avg IS NOT NULL
    AND (job_location LIKE '%UK%' OR job_location LIKE '%United Kingdom%')
ORDER BY
    salary_year_avg DESC
LIMIT 10;
```
Breakdown of findings:
- **Salary range**: There is a wide salary range of £180,000 - £120,000 within the top 10 highest-paying roles, indicating high potential; which may however skew the average.
- ** Diverse employers**: There is a range of employer types within the top 10, including finance & tech firms, but also other types of companies such as Shell and Hasbro.
- **Job title variety**: There are various job titles that fall under the data analysis umbrella within these top 10, including several Data Architect roles, as well as some research and engineering positions.

### 2. Skills for top-paying jobs:
To understand which skills are required for the highest-paying roles, I joined the skills data to the job postings table, to evaluate which skills employers value most for their roles.

```sql
WITH top_paying_jobs AS (
    SELECT
        job_id,
        job_title,
        job_location,
        salary_year_avg AS salary,
        company_dim.name AS company_name
    FROM
        job_postings_fact
    LEFT JOIN company_dim
    ON job_postings_fact.company_id = company_dim.company_id
    WHERE
        job_title_short = 'Data Analyst'
        AND job_schedule_type = 'Full-time'
        AND salary_year_avg IS NOT NULL
        AND (job_location LIKE '%UK%' OR job_location LIKE '%United Kingdom%')
    ORDER BY
        salary_year_avg DESC
    LIMIT 10
)

SELECT
    top_paying_jobs.*,
    skills
FROM 
    top_paying_jobs
INNER JOIN skills_job_dim ON top_paying_jobs.job_id = skills_job_dim.job_id
INNER JOIN skills_dim ON skills_job_dim.skill_id = skills_dim.skill_id
ORDER BY
    salary DESC;
```

Breakdown of findings:
- **SQL & Python**: Highly sought data analysis skills, both appearing in 5 of the top 10 postings.
- **Excel**: Also remains valued, appearing in 3 of the job postings.
- **BI tools**: Tools such as Power BI, Looker, and Tableau also remain present, appearing twice each in the top-10 jobs list.

### 3. In-demand skills for data analysts:
For a better understand of each skill's demand, this query identifies those most frequently sought in job postings, filtered to the relevant jobs we are focussed on.

```sql
SELECT
    skills,
    COUNT(skills_job_dim.job_id) AS demand_count
FROM
    job_postings_fact
INNER JOIN skills_job_dim ON job_postings_fact.job_id = skills_job_dim.job_id
INNER JOIN skills_dim ON skills_job_dim.skill_id = skills_dim.skill_id
WHERE
    job_title_short = 'Data Analyst'
    AND job_schedule_type = 'Full-time'
    AND (job_location LIKE '%UK%' OR job_location LIKE '%United Kingdom%')
GROUP BY
    skills
ORDER BY 
    demand_count DESC
LIMIT 5;
```
Breakdown of findings:
- **SQL** & **Excel** are fundamental, with many jobs requiring these foundational skills over more specific tools.
- **Power BI**, **Python**, and **Tableau** are also all commonly mentioned, highlighting how crucial data visualisation is to analysis.

### 4. Skills based on salary
Exploring the average salaries for different skills will detail which ones are the highest-paying for data analysts.

```sql
SELECT
    skills,
    COUNT(skills_job_dim.job_id) AS demand_count,
    ROUND(AVG(salary_year_avg),-3) AS skill_avg_salary
FROM
    job_postings_fact
INNER JOIN skills_job_dim ON job_postings_fact.job_id = skills_job_dim.job_id
INNER JOIN skills_dim ON skills_job_dim.skill_id = skills_dim.skill_id
WHERE
    job_title_short = 'Data Analyst'
    AND job_schedule_type = 'Full-time'
    AND (job_location LIKE '%UK%' OR job_location LIKE '%United Kingdom%')
    AND salary_year_avg IS NOT NULL
GROUP BY
    skills
ORDER BY 
    skill_avg_salary DESC
LIMIT 10
```

Breakdown of findings:
- Skills listed are more **specific**: The skills associated with the highest paying roles are more obscure, such as specific Python libraries or Machine Learning platforms, and are often requested only in one posting.
- These are **not** the most **optimal** skills to learn: Despite associated with the highest pay, the singular listings of these skills does necessarily not make them effective ones to learn.

### 5. Most optimal skills to learn

To identify the most optimal skills to learn, we evaluate their demand in conjunction with their associated pay.
For a more realistic viewpoint of skill demand, we add a condition to include skills mentioned in 10+ job positngs
```sql
SELECT
    skills,
    COUNT(skills_job_dim.job_id) AS demand_count,
    ROUND(AVG(salary_year_avg),-3) AS skill_avg_salary
FROM
    job_postings_fact
INNER JOIN skills_job_dim ON job_postings_fact.job_id = skills_job_dim.job_id
INNER JOIN skills_dim ON skills_job_dim.skill_id = skills_dim.skill_id
WHERE
    job_title_short = 'Data Analyst'
    AND job_schedule_type = 'Full-time'
    AND (job_location LIKE '%UK%' OR job_location LIKE '%United Kingdom%')
    AND salary_year_avg IS NOT NULL
GROUP BY
    skills
HAVING
    COUNT(skills_job_dim.job_id) > 10
ORDER BY 
    skill_avg_salary DESC
LIMIT 10
```
Breakdown of final findings:
- **SQL** leads demand: Listed in 31 applicable job postings, this tool is fundamental for data analysis, and ranks 2nd for average salary.
- **Python** leads salary: While mentioned less so than SQL, Python leads the salary ranking, remaining a critical skill for aspirational analysts.
- **Excel** remains key: Only just behind the leaders for demand and salary, it remains crucial that analysts have an undertsanding of the tool.
- **Tableau** not far behind: While of slightly lower demand than the key tools, Tableau remains important by its inclusion in this heavily filtered list.

# Conclusion
To summarise the findings, it is clear that key data manipulation tools such as SQL are vital to securing a Data Analyst position.

 Programming languages such as Python are also valued, which may be influenced by the broad scope of Data Analysis roles, which can often overlap with Data Science, where modelling often plays a key role.

 Other tools such as Excel, Tableau, and Power BI are also valued, however fall slightly behind the leaders for salary and demand; making them important to have knowledge of, but not necessarily the most optimal to learn in pursuit of the dream role.
